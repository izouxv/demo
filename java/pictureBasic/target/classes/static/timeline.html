  • 
    2017年9月2，心血来潮，从网上找了很久开源图床，但是没有自己满意的，之后就诞生此开源图床

    更久前，自己写 Markdown 笔记，笔记图片管理一直是我很头疼的事情，网上也有一些免费的图床，但是怕丢资源，一直也没用，总想着时间自己写一个图床
